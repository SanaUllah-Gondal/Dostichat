#  DostiChat

A safe, anonymous platform for people in India and Pakistan to connect via text and video chat.

## Features
-  Text chat with interest-based matching
-  WebRTC video calling (P2P, encrypted)
-  Light/dark mode
-  Moderation & reporting

## Tech Stack
- PHP (backend + signaling)
- JavaScript + WebRTC (frontend)
- HTML/CSS
